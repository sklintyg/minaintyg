Skicka ditt intyg till en eller flera mottagare

Från den här sidan kan du välja att skicka ditt intyg elektroniskt till en eller flera mottagare. Om du saknar din mottagare i listan beror det på att mottagaren inte kan ta emot elektroniska intyg. Du kan då ladda ner / skriva ut intyget och skicka det per post istället.

Myndighet som mottagare

Mina intyg kan aldrig garantera hur de tillgängliga mottagarna för ett intyg hanterar sekretessmarkerade personuppgifter eller om de följer Skatteverkets vägledning för hur sekretessmarkerade personuppgifter ska hanteras. Myndigheter ska ha särskilda rutiner för hantering av sekretessmarkerade personuppgifter.
