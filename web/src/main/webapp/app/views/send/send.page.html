Skicka ditt intyg till en eller flera mottagare

Från den här sidan kan du välja att skicka ditt intyg elektroniskt till en eller flera mottagare. Om du saknar din mottagare i listan beror det troligen på
att mottagaren inte kan ta emot elektroniska intyg. Du kan då ladda ner / skriva ut intyget och skicka det per post istället.
