!1 Integration med Försäkringskassan: Markera intyg som hämtat.

När Försäkringskassan sätter status på ett intyg, förväntar de sig följande svar:
|Markera intyg som hamtat|
|personnr|intyg|svar?|kommentar|
|121212-1212|intyg-6|[ERROR] - There is no certificate 'intyg-6' for user '121212-1212'.|försök att markera fel intyg för en invånare|
|121212-1212|intyg-999|[ERROR] - There is no certificate 'intyg-999' for user '121212-1212'.|försök att markera ett okänt intyg för en invånare|
|999999-9999|intyg-6|[ERROR] - There is no certificate 'intyg-6' for user '999999-9999'.|försök att markera ett intyg för en okänd invånare|
|121212-1212|intyg-1|ok|markering av ett befintligt intyg|
